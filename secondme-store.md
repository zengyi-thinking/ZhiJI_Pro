## 应用简介

知机 Pro 是一款具备情感理解能力的多模态 AI 聊天伴侣。不同于传统聊天机器人，知机 Pro 内置了 5 个专业情绪 Agent（理性、共情、成长、安全、行动），能够从多维度理解和回应用户的情感需求。

## 应用描述

### 核心功能

1. **多模态对话**
   - 支持文字、图片、语音输入
   - 实时流式响应体验
   - 智能上下文理解

2. **情感多智能体系统**
   - 理性 Agent：逻辑分析，客观建议
   - 共情 Agent：情感共鸣，温暖陪伴
   - 成长 Agent：启发思考，促进发展
   - 安全 Agent：风险评估，边界守护
   - 行动 Agent：具体建议，实践指引

3. **记忆管理**
   - 长期记忆存储
   - 语义向量检索
   - 个性化对话体验

4. **图片理解**
   - 场景识别与分析
   - 视觉内容摘要
   - 图文结合理解

5. **情感表达**
   - AI 生成情感卡片
   - 语音合成输出
   - 多样化表达方式

### 技术特点

- 采用 GPT-5 系列模型
- 向量嵌入记忆检索
- 内容安全审核
- Docker 容器化部署

## 使用说明

1. **开始对话**：输入文字或上传图片，与 AI 交流
2. **查看记忆**：浏览 AI 记住的对话历史
3. **语音互动**：录音转文字，或让 AI 语音回复
4. **生成卡片**：获取 AI 生成的情感主题图片

## 部署信息

- 后端服务：Node.js + Express + TypeScript
- 部署平台：Zeabur
- 健康检查：`GET /health`
- API 前缀：`/api/`

## 隐私说明

- 用户数据存储在服务端，用于记忆管理
- 支持 API Key 鉴权
- 建议使用 HTTPS 部署
